=> This Game can only be played by 2 Players.
=> In this, there are 2 boxes which displays a random number between 0 to 100 and there is a generate button which generates random numbers in both boxes.
=> On selecting generate button which displays the number. After that select Add Score button to increment the scores.
=> There are also 2 score pannels for each players.
=> The score of Player 1 increments whenever the generated number is greater than the genrated number of Player 2 on clicking the Add score button.
=> The score of Player 2 increments whenever the generated number is greater than the genrated number of Player 1 on clicking the Add score button.
=> If the Numbers are same, on clicking the Add Score button, the score remains the same.
=> The Player who first scores 10 wins the game.
